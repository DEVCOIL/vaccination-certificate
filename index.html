COVIDGET

Мобильное приложение COVIDGET (Ковиджет) создано для более простого использования исключений из мер эпидемиологической безопасности населением России и работает только с персональными QR-кодами вакцинации от COVID-19 России.

Мобильное приложение COVIDGET (Ковиджет) не собирает и не хранит ваши персональные данные. Все результаты сканирования вашего персонального QR-кода остаются только на вашем устройстве и никуда не передаются.

При скачивании и спользовании этого приложения Вы обязуетесь соблюдать нормативные акты о защите данных физических лиц, использовать приложение и обрабатывать личные данные только с той целью, чтобы лица, которые вакцинированы против Covid-19, могли бы использовать исключения из мер эпидемиологической безопасности.
