СЕРТИФИКАТ ВАКЦИНАЦИИ

Мобильное приложение СЕРТИФИКАТ ВАКЦИНАЦИИ создано для более простого использования исключений из мер эпидемиологической безопасности населением России и работает только с персональными QR-кодами вакцинации России.

Мобильное приложение СЕРТИФИКАТ ВАКЦИНАЦИИ не собирает и не хранит ваши персональные данные. Все результаты сканирования вашего персонального QR-кода остаются только на вашем устройстве и никуда не передаются.

При скачивании и спользовании этого приложения Вы обязуетесь соблюдать нормативные акты о защите данных физических лиц, использовать приложение и обрабатывать личные данные только с целью использования исключений из мер эпидемиологической безопасности.
